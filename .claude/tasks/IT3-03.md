# IT3-03: Pre-built Signature Templates

**Status:** IN PROGRESS
**Depends on:** IT3-01
**Wave:** 2

## Scope

Create a library of 5-6 curated signature templates that users can start from:
- Template definitions in `lib/templates/signatureTemplates.ts`
- Templates include:
  - Creative Professional (warm gold, horizontal layout)
  - Corporate Classic (blue, stacked layout)
  - Minimalist (gray, compact layout)
  - Bold & Colorful (purple, horizontal layout)
  - Tech Founder (teal, horizontal layout)
  - Freelancer (slate, stacked layout)
- `TemplateCard` component to display template preview
- Templates page (`/templates`) with grid layout
- "Use This Template" button flow:
  1. User clicks template
  2. Create new signature in store with template data
  3. Set as active signature
  4. Redirect to `/editor`

## Files

**Create:** `lib/templates/signatureTemplates.ts`, `components/templates/TemplateCard.tsx`, `app/templates/page.tsx`
**Modify:** Existing store actions used (no modifications needed)
**Tests:** `__tests__/lib/templates/signatureTemplates.test.ts`, `__tests__/components/templates/TemplateCard.test.tsx`

## Acceptance

- 5-6 template definitions with realistic placeholder data
- Each template has distinct visual identity (color + layout combo)
- TemplateCard renders template preview accurately
- Templates page displays grid of templates
- "Use This Template" creates signature, sets active, and redirects
- Tests cover template data structure and component rendering

# IT2-07: Copy-to-Clipboard

**Status:** PENDING
**Depends on:** IT2-06
**Wave:** 4

## Scope

Copy generated HTML as rich text so users can paste formatted signatures into Gmail/Outlook.

- Use `navigator.clipboard.write()` with `text/html` MIME type via `ClipboardItem`
- Fallback to `navigator.clipboard.writeText()` for older browsers
- `CopyButton` component shows "Copied!" feedback for 2 seconds

# IT2-03: App Branding & UI Foundation Components

**Status:** PENDING
**Depends on:** IT2-01
**Wave:** 2

## Scope

Restyle app with Technology for Artists branding: clean, minimal, whitespace-heavy, neutral/white palette, modern sans-serif. Replace indigo accent theme. Migrate `<a>` tags to `next/link`.

Create reusable UI components:
- `components/ui/Button.tsx` — variants: primary (gray-900 bg), secondary (white bg, border), ghost
- `components/ui/Input.tsx` — labeled text input with error/helper text, accessible (htmlFor/aria)
- `components/ui/Select.tsx` — labeled select dropdown
- `components/ui/Card.tsx` — card container
- `components/ui/SectionHeading.tsx` — section title + description

## Files

**Modify:** `app/layout.tsx`, `app/globals.css`, `app/page.tsx`
**Create:** `components/ui/Button.tsx`, `components/ui/Input.tsx`, `components/ui/Select.tsx`, `components/ui/Card.tsx`, `components/ui/SectionHeading.tsx`
**Tests:** `__tests__/components/ui/Button.test.tsx`, `__tests__/components/ui/Input.test.tsx`

## Acceptance

- App uses neutral/white palette, no indigo
- Navigation uses `next/link`
- All UI components render correctly with accessible markup
- Button/Input tests pass

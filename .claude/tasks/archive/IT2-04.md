**Status:** DONE

## Scope

Build the multi-section editor form that reads/writes to the Zustand store.

Components:
- `components/editor/SignatureForm.tsx` — orchestrator
- `components/editor/SignatureNameField.tsx` — signature name
- `components/editor/PersonalInfoSection.tsx` — fullName, jobTitle, company
- `components/editor/ContactInfoSection.tsx` — email, phone, website
- `components/editor/SocialLinksSection.tsx` — add/remove social + custom links
- `components/editor/BrandingSection.tsx` — color swatches + layout picker

Store updates:
- Add `activeSignatureId`, `setActiveSignatureId`, `getActiveSignature` to signature slice
- Update persist `partialize` to include `activeSignatureId`

**Status:** IN PROGRESS

## Scope

Expand `lib/schemas/signature.schema.ts` to add:
- Social links: Array of `{ platform: enum, url: string }` — 12 platforms (max 10)
- Custom links: Array of `{ label: string, url: string }` (max 5)
- Branding: `{ colorPresetId: string, layoutPresetId: 'horizontal' | 'stacked' | 'compact' }`
- 8 curated color presets with primary + accent hex colors
- 3 layout presets

All new fields use `.default()` for backward compatibility.

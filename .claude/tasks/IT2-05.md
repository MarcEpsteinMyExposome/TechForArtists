# IT2-05: Live Signature Preview

**Status:** DONE
**Depends on:** IT2-01, IT2-03
**Wave:** 3
**Completed:** 2026-02-06

## Scope

React component that renders a visual preview of the active signature, updating in real-time. Uses Tailwind for styling (not the email HTML output).

Components:
- `components/editor/SignaturePreview.tsx` — container
- `components/editor/preview/HorizontalLayout.tsx` — photo left, info right
- `components/editor/preview/StackedLayout.tsx` — photo top, info below
- `components/editor/preview/CompactLayout.tsx` — text only, no photo

Layout specs:
- Horizontal: `[Photo] Name / Title | Company / email | phone / website / [social links]`
- Stacked: Centered vertical stack
- Compact: Text-only, pipe-separated

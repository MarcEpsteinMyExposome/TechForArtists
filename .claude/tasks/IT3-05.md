# IT3-05: Custom Color Picker

**Status:** PENDING
**Depends on:** IT3-03
**Wave:** 3

## Scope

Allow users to choose arbitrary hex colors beyond the 8 color presets:
- Add "Custom" option to the color swatch section in `BrandingSection`
- HTML color input (`<input type="color">`) or hex text field for custom color entry
- Store custom colors in signature branding
- May require schema update to support `customPrimaryColor` and `customAccentColor` fields
- Presets remain available as quick-pick options
- Custom colors integrate seamlessly with existing signature generation

## Files

**Modify:** `lib/schemas/signature.schema.ts` (add customPrimaryColor/customAccentColor if needed), `components/editor/BrandingSection.tsx`
**Create:** `components/editor/ColorPicker.tsx` (optional, if building custom UI)
**Tests:** `__tests__/lib/schemas/signature.test.ts`, `__tests__/components/editor/BrandingSection.test.tsx`

## Acceptance

- Users can select "Custom" color option
- Color picker or hex input allows arbitrary color entry
- Custom colors stored in signature data
- Signature preview renders with custom colors
- HTML generation uses custom colors correctly
- Presets still work as before
- Tests cover custom color validation and rendering
